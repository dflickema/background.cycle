Background Cycle example

This page is an example usage of background.cycle.js. It uses the following code: 

                
        $("body").backgroundCycle({
            imageUrls: [
                'res/img/bg1.jpg',
                'res/img/bg2.jpg',
                'res/img/bg3.jpg'
            ],
            fadeSpeed: 2000,
            duration: 5000,
            backgroundSize: SCALING_MODE_CONTAIN
        });
